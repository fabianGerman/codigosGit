C++

C++ es un lenguaje de programacion diseñado en 1979. La intencion de su creacion fue extender al lenguaje de programacion C mecanismos que permiten la manipuclacion de objetos. En ese sentido, desde el punto de vista de los lenguajes orientados a objetos C++ es un lenguaje hibrido


Actualmente existe un estandar , denominado ISO C++, al que se han adherido la mayoria de de los fabricantes de compiladores mas modernos. Existen tambien algunos interpretes, tales como ROOT

Java

Java es un lenguaje de programacion y una plataforma informatica comercializada por primera vez en 1995. Hay muchas aplicaciones y sitios web que no funcionaran a menos que tenga Java instalado y cada dia se crean mas. Java es rapido, es seguro y fiable. Desde portatiles, hasta centros de datos, desde consolas, para juegos hasta super computadoras, Java esta en todas partes
